# Weekly Planning System

## Context
Effective personal productivity requires intentional planning and organization. This prompt helps create a structured weekly planning system.

## Prompt
```
Help me create a comprehensive weekly planning system that fits my specific needs and circumstances.

About me:
- [Describe your role/job/life situation]
- [Mention key ongoing projects or responsibilities]
- [Note any productivity challenges you face]
- [Describe your preferred work style: digital/analog, visual/text, etc.]

I want my weekly planning system to help me:
- [List 2-3 primary goals for your planning system]

Please provide:
1. A structured template for weekly planning with specific sections
2. A daily planning framework that complements the weekly system
3. Strategies to prioritize tasks effectively
4. Methods to track progress and review accomplishments
5. Suggestions for tools or apps that would support this system
6. Tips for maintaining the system consistently
7. A process for weekly review and adjustment

I prefer [high detail/minimalist] approaches and need [flexibility/structure].
```

## Notes
- Adjust the level of structure based on your preference
- For busier schedules, emphasize time-blocking techniques
- Consider requesting integration with existing systems you already use
